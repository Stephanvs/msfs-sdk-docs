Engines Configuration File

  • Introduction
  • Sections

Introduction

The engines configuration file is an optional file; the sections described here may be defined in the aircraft.cfg as well. If used, it should be named engines.cfg and located in the root directory of the aircraft, next to the aircraft.cfg.

This documentation is not yet complete. For now, please look at the ESP documentation‘s Aircraft Configuration File for missing sections’ parameters. Most (if not all) of the parameters described there are valid for Microsoft Flight Simulator.

Sections

[VERSION]

This version section provides information on version of the configuration file. The major version must be 1 or greater and equal to the version of the aircraft.cfg.

Property Description Type Unit Range Mandatory
major Major CFG file version number Integer Number >= 1 Yes
minor Minor CFG file version number Integer Number >= 1 Yes

[GENERALENGINEDATA]

Every type of aircraft, even a glider, should have this section in the engines.cfg file. Basically, this section describes the type of engine, the number of engines, where the engines are located, and a fuel flow scalar to modify how much fuel the engine requires to produce the calculated power.

Property Description Type Unit Range Mandatory
engine_type Integer that identifies what type of engine is on the aircraft. 0 = piston, 1 = Jet, 2 = None, 3 = Helo-turbine, 4 = Rocket (not supported) 5 = Turboprop. Integer Enum Between 0 and 5 Yes
fuel_flow_scalar Scalar for modifying the fuel flow required by the engine(s). A value of less than 1.0 causes a slower fuel consumption for a given power setting, a value greater than 1.0 causes the aircraft to burn more fuel for a given power setting. Float No unit > 0 Yes
min_throttle_limit Defines the minimum throttle position (percent of max). Normally 0 for piston aircraft and -0.25 for turbine airplane engines with reverse thrust. Float Percentage Between -1 and 1 Yes
master_ignition_switch 1=Available, 0=Not Available (default). If available, this switch must be on for the ignition circuit, and thus the engines, to be operable. Turning it off will stop all engines. Integer Enum 0 or 1 No
starter_type Stater type 0=Electric, 1=Manual Integer Enum 0 or 1 No
max_contrail_temperature Ambient temperature in which engine vapor contrails will turn on. The default value is about -39 degrees celsius for turbine engines. For piston engines, the contrail effect is turned off unless a temperature value is set here. Float Celsius degrees No
Engine.i Offset of the engine i from the datum reference point. z (longitudinal), x (lateral), y (vertical) distance from datum reference. Each engine location specified increases the engine count (maximum of four engines allowed i.e i is between 0 and 3) 1D table, 3 floats Feet Yes
ThrustAnglesPitchHeading.i Thrust pitch and heading angles for engine number i 1D table, 2 floats < 0 for pitch down, > 0 for heading right Degrees No

[PISTON_ENGINE]

A piston engine’s power can be determined through a series of equations that represent the Otto cycle of a four-stroke piston engine, multiplied by the number of pistons available. This section contains all the information needed to be able to determine how much power the engines are capable of producing.
