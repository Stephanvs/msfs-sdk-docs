Introduction

The Microsoft Flight Simulator low level API exposes 13 functions and a few structures that allow C/C++ gauges to be drawn into the provided in-game bitmap. All the functions use a ctx parameter of type FsContext which is provided as an argument of the gauge callback.

Structures

The following structures are used by the low level API:

FsColor

The FsColor structure represents a typical color through its Red, Green, Blue and Alpha components.

Members

  • The rgba field is an array of 4 floats that can be used to define the color components with an index.
  • Individual components can be accessed through the r, g, b and a fields.

FsPaint

The FsPaint structure represents a color, gradient or image pattern used to fill shapes.

Members

  • The xform field defines the transformation of the pattern. See below for a more detailed explanation.
  • The extent field defines the size of the pattern. See below for a more detailed explanation.
  • The radius field defines the radius of the rounded corners of a box gradient.
  • The feather field defines the blurriness of the border of a box gradient.
  • The innerColor field defines the inner color of a gradient.
  • The outerColor field defines the outer color of a gradient.
  • The image field defines the handle to the image used for this pattern.

FsCompositeOperation

The FsCompositeOperation represents a composite operation used to combined a source pixel and a destination pixel into a final pixel.

Members

The four members srcRGB, dstRGB, srcAlpha and dstAlpha are used to compute the final pixel as follows:

  • (R,G,B)final = srcRGB * (R,G,B)source + dstRGB * (R,G,B)destination
  • (A)final = srcAlpha * (A)source + dstAlpha * (A)destination

With the following values available for each member:

  • FS_ZERO
  • FS_ONE
  • FS_SRC_COLOR
  • FS_ONE_MINUS_SRC_COLOR
  • FS_DST_COLOR
  • FS_ONE_MINUS_DST_COLOR
  • FS_SRC_ALPHA
  • FS_ONE_MINUS_SRC_ALPHA
  • FS_DST_ALPHA
  • FS_ONE_MINUS_DST_ALPHA
  • FS_SRC_ALPHA_SATURATE

FsScissor

The FsScissor structure represents a rectangular region, potentially transformed, that can be used as a mask.

Members

  • The xform field defines the transformation of the scissor region.
  • The extent field defines the size of the scissor region.

FsVertex

The FsVertex represents a typical 2D vertex.

Members

  • The x and y fields define the vertex location.
  • The u and v fields define the vertex texture coordinates.

FsPath

The FsPath structure represents a shape to be drawn.

Members

  • The first field is ignored for now.
  • The count field is ignored for now.
  • The closed field is ignored for now.
  • The nbevel field is ignored for now.
  • The fill field defines the vertices that delimit the shape to be filled.
  • The nfill field defines the number of vertices in the fill field.
  • The stroke field defines the vertices that delimit the shape to be stroke.
  • The nstroke field defines the number of vertices in the stroke field.
  • The winding field is ignored for now.
  • The convex field defines the convexity of the shape: 0 for concave, any other value for convex.

Additional information

  • The xform field which defines a transformation in some of the structures above is a lightweight version of 3x3 transformation matrix. If you consider the following matrix:
    >m11 m12 m13
    m21 m22 m23
    m31 m32 m33

    The xform field is defined as an array of 6 floats: [ m11, m21, m12, m22, m13, m23 ].

  • The extent field which defines a size in some of the structures above is an array of 2 floats: [ width, height ]

Functions

The following functions are available:

fsRenderCreate

The fsRenderCreate function creates a new rendering context in the game.

Syntax

int fsRenderCreate(
          FsContext ctx
      )

Parameters

  • ctx: a value of type FsContext provided as an argument of the gauge callback.

Return Values

This function returns 1 if it is successful, or 0 if it failed.

fsRenderCreateTexture

The fsRenderCreateTexture creates a new texture.

Syntax

int fsRenderCreateTexture(
          FsContext ctx,
          int type,
          int w,
          int h,
          int imageFlags,
          const unsigned char* data,
          const char* debugName
      )

Parameters

  • ctx: a value of type FsContext provided as an argument of the gauge callback.
  • type: the texture type - 1 for Alpha only, 2 for RGBA.
  • w: the width of the texture to create.
  • h: the height of the texture to create.
  • imageFlags: a combination (logical or) of the following values:
    • 1 to generate mipmaps automatically
    • 2 to repeat the image in the X direction
    • 4 to repeat the image in the Y direction
    • 8 to flip the image on the Y axis
    • 16 to indicate that the image data has premultiplied alpha
    • 32 to specify a Nearest instead of a Linear interpolation
  • data: the texture data.
  • debugName: the texture name for debugging purposes.

Return Values

This function returns a handle to the created texture (to be used by the image member of the FsPaint structure) if it is successful, or 0 if it failed.

fsRenderDeleteTexture

The fsRenderDeleteTexture deletes a texture.

Syntax

int fsRenderDeleteTexture(
          FsContext ctx,
          int image
      )

Parameters

  • ctx: a value of type FsContext provided as an argument of the gauge callback.
  • image: a handle to the texture to delete.

Return Values

This function always returns 0.

fsRenderUpdateTexture

The fsRenderUpdateTexture function updates part or all of a texture.

Syntax

int fsRenderUpdateTexture(
          FsContext ctx,
          int image,
          int x,
          int y,
          int w,
          int h,
          const unsigned char* data
      )

Parameters

  • ctx: a value of type FsContext provided as an argument of the gauge callback.
  • image: a handle to the texture to update.
  • x: the x-coordinate of the top left corner of the region to update.
  • y: the y-coordinate of the top left corner of the region to update.
  • w: the width of the region to update.
  • h: the height of the region to update.
  • data: the updated data.

Return Values

This function returns 1 if it is successful, or 0 if it failed.

fsRenderGetTextureSize

The fsRenderGetTextureSize function gets the specified texture size.

Syntax

int fsRenderGetTextureSize(
          FsContext ctx,
          int image,
          int* w,
          int* h
      )

Parameters

  • ctx: a value of type FsContext provided as an argument of the gauge callback.
  • image: a handle to the texture to get the size of.
  • w: a pointer to an int receiving the texture width.
  • h: a pointer to an int receiving the texture height.

Return Values

This function always returns 1.

fsRenderViewport

The fsRenderViewport function specifies the viewport to be used when drawing the gauge.

Syntax

void fsRenderViewport(
          FsContext ctx,
          float width,
          float height,
          float devicePixelRatio
      )

Parameters

  • ctx: a value of type FsContext provided as an argument of the gauge callback.
  • width: the width of the viewport.
  • height: the height of the viewport.
  • devicePixelRatio:

Return Values

This function does not return a value.

fsRenderCancel

The fsRenderCancel function cancels the current rendering commands.

Syntax

void fsRenderCancel(
          FsContext ctx
      )

Parameters

  • ctx: a value of type FsContext provided as an argument of the gauge callback.

Return Values

This function does not return a value.

fsRenderFlush

The fsRenderFlush function flushes (executes) the current rendering commands.

Syntax

void fsRenderFlush(
          FsContext ctx
      )

Parameters

  • ctx: a value of type FsContext provided as an argument of the gauge callback.

Return Values

This function does not return a value.

fsRenderFill

The fsRenderFill fills the specified shapes.

Syntax

void fsRenderFill(
          FsContext ctx,
          FsPaint* paint,
          FsCompositeOperationState compositeOperation,
          FsScissor* scissor,
          float fringe,
          const float* bounds,
          const FsPath* paths,
          int npaths
      )

Parameters

  • ctx: a value of type FsContext provided as an argument of the gauge callback.
  • paint: the pattern to use.
  • compositeOperation: the composite operation to use.
  • scissor: the scissor to use.
  • fringe: the size of the fringe used for anti-aliasing.
  • bounds: the boundaries of the shape. This is an array of 4 floats: [ xmin, ymin, xmax, ymax].
  • paths: the paths to draw.
  • npaths: the number of paths in paths.

Return Values

This function does not return a value.

fsRenderStroke

The fsRenderStroke function strokes the specified shapes.

Syntax

void fsRenderStroke(
          FsContext ctx,
          FsPaint* paint,
          FsCompositeOperationState compositeOperation,
          FsScissor* scissor,
          float fringe,
          float strokeWidth,
          const FsPath* paths,
          int npaths
      )

Parameters

  • ctx: a value of type FsContext provided as an argument of the gauge callback.
  • paint: the pattern to use.
  • compositeOperation: the composite operation to use.
  • scissor: the scissor to use.
  • fringe: the size of the fringe used for anti-aliasing.
  • strokeWidth: the width of the lines produced by the stroke.
  • paths: the paths to draw.
  • npaths: the number of paths in paths.

Return Values

This function does not return a value.

fsRenderTriangles

The fsRenderTriangles function renders the specified triangles.

Syntax

void fsRenderTriangles(
          FsContext ctx,
          FsPaint* paint,
          FsCompositeOperationState compositeOperation,
          FsScissor* scissor,
          const FsVertex* verts,
          int nverts
      )

Parameters

  • ctx: a value of type FsContext provided as an argument of the gauge callback.
  • paint: the pattern to use.
  • compositeOperation: the composite operation to use.
  • scissor: the scissor to use.
  • verts: the vertices that define the triangles.
  • nverts: the number of vertices in verts.

Return Values

This function does not return a value.

fsRenderClearStencil

The fsRenderClearStencil function clears the stencil buffer of the specified context.

Syntax

Parameters

  • ctx: a value of type FsContext provided as an argument of the gauge callback.

Return Values

This function does not return a value.

Syntax

void fsRenderDelete(
          FsContext ctx
      )

Parameters

  • ctx: a value of type FsContext provided as an argument of the gauge callback.
