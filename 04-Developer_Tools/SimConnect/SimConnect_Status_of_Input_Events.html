Status of Input Events

Notes

  • “Num_x” refer to the numpad keys when Num Lock is not active (e.g., “Num_9” maps to VK_PRIOR). For the numeric keys, please use VK_NUMPADx.
  • In order to use modifier keys (“Shift”, “Ctrl” and “alt”), it is required to explicitly use their left or right versions.

Known issues

  • Unsubscribing from an input event and pressing that key during cleanup raises an exception.
  • The client events outlive the SimConnect client session.
